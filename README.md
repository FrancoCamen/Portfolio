# Portfolio — Franco Camen

Portfolio personal desarrollado con React y Vite para presentar proyectos académicos y personales.

## Desarrollo local

```bash
npm install
npm run dev
```

## Compilación

```bash
npm run build
```

La salida de producción se genera en `dist/`.

## Publicación en GitHub Pages

El workflow incluido publica automáticamente el sitio cuando se envían cambios a la rama `main`.
En el repositorio de GitHub, configurar **Settings → Pages → Source** como **GitHub Actions**.
